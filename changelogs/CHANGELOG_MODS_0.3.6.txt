			Note that the mod has changed names! I'll be updating the pages shortly, but the new version is called "Storage for ComputerCraft" rather than "Refined Storage for ComputerCraft" because the new version adds experimental support for Applied Energistics 2 as well. All the internal ids etc for the mod have also changed, but I did a magic such that you can still safely upgrade older worlds without losing anything. This is a one-way trip, though: if you later try to swap back to 1.1.0 or older then the peripheral blocks will vanish and you'll have to craft them again.
			AE2 is an optional dependency -- you do not need to include it in your mod list if you don't want to. Conversely, however, Refined Storage is still a required dependency -- even if you only want to use AE2, you have to have RS installed if you want to use this mod. That will probably change in a future version.
			AutoStock is also included and supported for AE2, although since AE2 doesn't support fluid crafting it has fewer features. You can connect both RS and AE2 to the same computer if you like, but only when using your own programs -- AutoStock won't like it; keep it on separate systems.
